A function that adds two numbers.

Example

For param1 = 1 and param2 = 2, the output should be
add(param1, param2) = 3.

Input/Output

[time limit] 4000ms (php)
[input] integer param1

Guaranteed constraints:
-100 ≤ param1 ≤ 1000.

[input] integer param2

Guaranteed constraints:
-100 ≤ param2 ≤ 1000.

[output] integer

The sum of the two inputs.
